Overview

One very useful feature of the Character Editor is the Validator found in the main toolbar. This allows you validate either the current character, or all characters. Validation looks over the character's files for any problems: missing images, invalid dialogue (bad conditions, variables, etc.) and more. Before committing anything back to the repository, you should always validate your character. This ensures that your character is always at the minimum fully functional.

Validation Results

After validating one or all characters, you will be taken to the Results screen which will list all issues with the characters validated. The bottom left has a list of filters. If you are interested in seeing only a certain subset of issues, check and uncheck the appropriate filters here.
