Dialogue Editor

  • The "Respond to This" button now opens a dialogue editor in a popup, allowing you to more quickly go back to the first character after making a response.
  • Support for persistent markers has been added. A persistent marker will last between games and even sessions. To check a persistent marker, use the "Persistent Marker" condition rather than "Said Marker".
  • To reduce clutter in the line table, persistent markers, along with the old marker value and "per target" columns, have been placed underneath a dropdown button with a bookmark icon. This icon will appear solid for lines that use these options.
  • For the same reason, the other settings that used to appear under the "Show advanced" checkbox now appear under a ... dropdown.

Data Analyzer

  • Added some example queries.
  • Added some more data points for epilogue unlock conditions.

Pose Maker

  • Added a button to change the background color of the pose maker canvas, useful when working with light-colored sprites that blend in with the default background color.

General

  • A "Recent" list has been added to Character and Skin searches.
  • Marker data has been moved into editor.xml to consolidate non-game files. markers.xml will be deleted upon your next save.
  • Tags now display only as the tag key rather than the user-friendly display name (e.g. blue_eyes instead of Blue [blue_eyes]).
  • Added a section on the Create Sequence wizard to the "Pose Maker - Frame-by-Frame Animation" tutorial.
  • Added support for "best" AI level, which is slightly better than good.

Bug Fixes

  • Hidden cases now sort at the top.
  • Hidden cases are no longer included in line counts.
  • Fixed a crash when deleting a custom pose.
  • Sprites hidden by markers can no longer be selected in the canvas.
  • Fixed bug that would complain about failing to load characters when those characters are just empty folders.
  • Fixed issue where unique lines in meta.xml were being underreported.
  • Fixed issue where the first keystroke when changing text in a search field would clear the field but not add the key.
