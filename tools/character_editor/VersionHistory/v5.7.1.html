Pose Matrix Editor

  • Added clipboard support for copying rows, columns, or cells.
  • Added button to copy a code from one cell into the rest of the column.
  • Added arrow key navigation.
  • Added ability to duplicate a sheet.
  • For skins, added button to pull in columns for all poses that the main character uses.
  • Added indicator to skins indicating poses that exist in the main character but not the skin.
  • Added a button to open the character folder from the Pose Matrix Editor.

Bug Fixes

  • Fixed issues with the Pose Matrix editor in a skin saving and reading images from the main character directory.
  • Various minor Pose Matrix bug fixes.
