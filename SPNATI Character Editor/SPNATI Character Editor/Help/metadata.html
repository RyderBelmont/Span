2. Metadata
  • Overview
  • Fields

Overview

Metadata defines characteristics about the character, which is used for determining how other characters will react to yours, as well as the data that shows on the player selection screen.

Fields

  • Label: This is the name other characters will use when referring to your character.
  • First name: Character's first name during player selection. Not used anywhere else.
  • Last name: Character's last name during player selection. Not used anywhere else.
  • Default portrait: Which image to use on the player selection screen.
  • Gender: Character's gender. Characters react differently based on gender.
  • Size: For males, this is the size of their genitals. For females, this is the size of their chest. Characters react differently based on size.
  • AI: Determines how well the character plays. Bad AI will throw away random cards. Good AI will try to make the best moves. For player enjoyment's sake, you should try to balance AI and number of clothing layers. Bad AI with too few layers will lose too quickly too often, and good AI with too many layers will be frustrating to try to beat.
  • Rounds to finish: After a player has lost a hand while naked, they must masturbate. This value controls how many times the player has to click buttons before the character "finishes."
  • Writer: Who wrote the dialogue for the character
  • Artist: Who made the art for the character
  • Origin: Source material (game, anime, movie, etc.)
  • Height: How tall the character is. This data is currently useless.
  • Tags: A list of adjectives that define characteristics about your character. These allow other characters to target dialogue towards your character without targeting your character specifically.
    For instance, a character might react to any character that has shaved their pubic hair. By including a "shaved" tag, that character will automatically react to yours without needing to go back and edit that character.
    You can use anything for your tags, but it's important to be consistent. A tag used only by your character is not a very useful tag, nor is a tag that is not used in any other character's dialogue.
  • Description: Character bio used in the character selection screen.
