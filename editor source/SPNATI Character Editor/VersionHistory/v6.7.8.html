Dialogue

  • Adds functionality for forfeit operations in the Advanced Dialogue Options window. A given line can now manipulate stamina, forfeit timer, or forfeit redirects, as well as the heavy-masturbating state. See the documentation for more details.
  • The image preview now correctly shows custom poses exclusive to costumes.
  • The image preview now shows custom poses with the correct height.

Pose Matrix

  • Adds a checkmark button that marks the selected cells as imported.

Validator

  • Flags lines with weight 0, since weight 0 does not behave as expected.
  • Checks more robustly for lines with the whole line as {small}.
  • No longer warns for {!reset} at the end of a line, since it doesn't actually do anything bad per se.
  • No longer warns for Self Saying Marker, Self Saying Text, or Self Pose if the case is hidden and has negative priority.

Miscellaneous

  • Variable definitions have been moved into variables.xml in the opponents folder. This will allow mods to update available variables without requiring a new editor release.
  • Lines that check for the absence of a character/tag no longer add to the specific count of targeted/filtered lines toward that character/tag.
  • Lines that check for the absence of a character no longer count as targeted lines.
  • Lines that check "self" for tags no longer count as filtered lines.

Bug Fixes

  • Fixes a bug causing data recovery backups to never be deleted. This will take effect, on a per-character basis, the first time you save the character in this version, which may cause an unusually long time for that save to complete.
  • Fixes the Pose Maker sometimes corrupting image file paths.
  • Images are now properly considered to be uncompressed if they were compressed but then modified later.
  • Gives KKL more time to respond to commands, which should fix some importing issues.
