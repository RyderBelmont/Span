Metadata

  • Allows for setting a separate label to be used on just the character selection screen. (For example, Mercy can now be "Mercy" in the selection screen but still have all characters refer to her as "Angela".)

Miscellaneous

  • Tags are now stored in a separate "tags.xml" file, allowing them to be edited by a third party without causing merge conflicts.

Bug Fixes

  • No longer deletes all backups when creating a new character on top of a previously deleted one.
  • Opening a case that targets a nonexistent character no longer creates a blank character with that name.
  • Allows percentage-based font sizes to increase over 100%.
  • Fixes a bug preventing the Configure Game dialog from opening during a re-sort or poll.
  • The "Respond" button no longer creates cases in the old, pre-5.2 format.
