Dialogue Editor

    "Respond to This" button now opens a dialogue editor in a popup, allowing you to more quickly go back to the first character after making a response.

General

  • A "Recent" list has been added to Character and Skin searches.
  • Marker data has been moved into editor.xml to consolidate non-game files. markers.xml will be deleted upon your next save.
  • Tags now display only as the tag key rather than the user-friendly display name (e.g. blue_eyes instead of Blue [blue_eyes]).
  • Added a section on the Create Sequence wizard to the "Pose Maker - Frame-by-Frame Animation" tutorial.

Data Analyzer

  • Added some example queries.

Bug Fixes

  • Fixed issue with the Secret and Hide Details collectible tooltips being reversed.
  • Hidden cases now sort at the top.
  • Hidden cases are no longer included in line counts.
  • Fixed a crash when deleting a custom pose.
  • Fixed a crash when clicking the Add button for dialogue while no case is selected for Finished and Game Over cases.
