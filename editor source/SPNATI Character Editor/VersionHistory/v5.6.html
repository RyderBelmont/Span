General

  • A warning banner will display when viewing characters saved in later versions of the editor.
  • Tag definitions have been moved into tag_dictionary.xml in the opponents folder. This will allow mods to update available tags without requiring a new editor release.

Dialogue

  • A "Prompt" category has been added when calling out situations. This is intended for lines where the character is asking the others a question, making a request, or otherwise soliticing feedback.

Alternate Skins

  • The status field is now visible.

Miscellaneous

  • Added a "Goto Line" button to jump to the line with the mispelled word in the dialogue editor.
  • Added Writer and Artist criteria to the Data Analzyer

Bug Fixes

  • Creating a response to a "Selected" case will now create an "Opponent Selected" case rather than just "Selected."
  • Responding to a case like (target: Florina, 0 masturbating, 0 finished) will now create a "must masturbate (self, first to do so)" response instead of "must masturbate (self)"
  • Fixed the Delete button in the Wardrobe Editor. Previously it only allowed deleting layers on a released character when it should have been the opposite.
  • The Response and Call Out buttons are now disabled for Finishing (Self) cases because no other character can say anything during these lines.
  • The same thing applies to Hidden cases.
  • Fixed issue where pating lines and overwriting existing ones would look like it overwrote them but didn't actually.
  • Validator checks for "saying text" that never gets said again. Previously this only checked the pre-5.2 condition format. For performance reasons, these checks only run if the other character has been loaded in the editor during the session, however.
  • Fixed issue where custom tags wouldn't be saved.
  • Tweaked the wording of the "Last Place Loses Hand" recipe to clarify that it can happen even on that player's first loss.
