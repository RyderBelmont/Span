Validator

  • Restores a large number of validation rules that weren't working.
  • Now checks for issues within AND/OR tabs as well.
  • Flags conditions with a count of "0+", which is meaningless and was usually meant to be a count of zero.
  • Flags Self Saying Marker, Self Saying Text, and Self Pose conditions, which never work.
  • Flags redundant use of ifMale in an already gendered case.

  • Clarifies if a clothing classification is only invalid due to being incorrectly capitalized.

Advanced
