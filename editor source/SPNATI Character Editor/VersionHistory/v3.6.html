Dialogue Editor

  • New variables mainly intended for Variable Test (+) conditions. See merge request !1337 for more details.
  • Variable Tests now allow more comparison operators than just ==.

Poses Importer

  • Poses can now have transparency levels assigned to different body and clothing parts. For instance, you can make a partially transparent skirt or hide your model's legs when importing from Kisekae.
  • Use the More... button for a particular pose to bring up the Transparency selection screen. This information is saved with the pose list.

Banter Wizard

  • Performance optimizations - opposing characters' lines will be loaded only when clicked, vastly improving the initial load. For the same reason, tags are no longer automatically scanned. A downside is that characters with no lines will now appear in the list.
  • Selecting an opponent's line now displays the corresponding preview image.

Bug Fixes

  • Icons that would appear as □ now appear properly.
  • Fixed crash when using custom poses where a sprite's file did not exist.
