Metadata

  • Adds the option to add a name to the default costume.

Dialogue

  • Adds "Not Tag" and "Tags (Advanced)" dialogue conditions.
  • Adds the "@" operator for markers, which checks if a marker is in a given range.
  • Adds a "Refresh Images" button, which refreshes the pose dropdown. Use this after adding new poses - previously this required restarting the CE entirely.

Costumes

  • Allows costumes to skip layers by selecting the new clothing type "skip". Please consult more experienced users before trying to do this!

Pipelines

  • Adds Arithmetic, Compare, Concatenate, and Substring nodes for advanced operations.
  • Adds a row output to the Cell node which can be used as input for a Cell Reference or one of the above.

Configuration

  • Adds the option to only show custom poses in the pose dropdown, useful for certain characters.

Miscellaneous

  • Adds support for the experimental KKL v105.5a10.

Bug Fixes

  • Allows invalid stages to be deselected, if they were somehow selected.
  • Custom pose assets stored in subfolders now count towards the file size total.
  • Cross-stage custom poses are now counted correctly in the pose count.
