Dialogue

  • Adds a "~blank~" variable that expands to nothing. Use this for intentionally blank lines, instead of leaving them blank.
  • Adds a new "AI Opponent" filter to more easily specify opponents that are not the player.
  • Adds a "Disable Case" menu option in the Dialogue Tree's Config menu (gear icon), to temporarily disable a case entirely. This is different than Hidden cases because Hidden cases still run in the background, which could cause undesired behavior.
  • Adds an option to select text size as a direct attribute of a line, instead of just a custom style.
  • Adds a few more items to the shortcut menus.
  • Shrinks the font size of the preview speech bubble slightly, to match a similar change to the game.

Costumes

  • Removes the rule that event costumes cannot be set to online, due to changes in how event costumes are handled.
  • Replaces spaces with underscores in the folder name when creating a new costume.

Collectibles

  • Added an "unlisted" option for status. This could basically be accomplished already by making the collectible Secret, but this method may be more intuitive.

Advanced

  • Added an option to globally reduce a character's text size. Please only use this as a last resort.

Pose Maker

  • Custom poses for the base character now automatically work for alternate costumes, as long as corresponding versions of all referenced images are present in the costume folder. Previously you would have to copy over any custom poses you wanted to use for each alternate costume. This feature is activated the first time you save the base character in v6.6.2 or later.

Validator

  • Flags completely blank lines, in case they were created by accident. If a blank line is intentional, use ~blank~.
  • Flags lines that use ~name~ or ~target~ in cases that don't have a target (hand lines, self strip, etc).
  • Flags cases with empty OR tabs, as that invalidates any conditions in the other AND/OR tabs.

Miscellaneous

  • Lines toward the human player no longer count toward the total amount of targeted lines.
  • Removes redundant conditions from some Recipes.
  • Removes the redundant tag list from behaviour.xml. The issue with the SPNATI Utilities Bot has been resolved.

Bug Fixes

  • Fixes an error with global dialogue layering in the Advanced tab. If you intended for the dialogue to go over your character's image, re-select "over", which will now actually work.
  • Custom cross-stage poses now correctly count their component images as used in the Validator.
  • Fixes a crash when opening the Pose Matrix if the entire Stage 0 row is empty.
