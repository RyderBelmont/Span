Pose Matrix

Introducing the Pose Matrix tab. This tool provides another optional method for managing Kisekae pose codes as an alternative to pose lists, templates or external spreadsheets. Features include:
  • Auto-saving to poses.xml whenever character is saved.
  • Data recovery. Like other .xml files, the data recovery tools will backup and restore the pose matrix.
  • Spreadsheet-like display for an easy at-a-glance view to see what poses haven't been created yet.
  • Support for importing entire lineups of a character's strip sequence. For a single pose (ex. "nervous"), line up all stages side by side in Kisekae and import them all at once into the pose matrix.
  • Support for exporting lineups. Pick a pose column and use the Export Lineup button to load that poses strip sequence into Kisekae.
  • Templating support. Like templates, you can optionally assign a base appearance code, stage codes for clothing, and poses for each emotion, and it'll combine these into the proper clothing + emotion for each stage when importing. This lets you update a pose across all stages or a stage's clothing across all poses without manually updating each one.
  • Like Pose Lists, provides buttons for bulk reimporting of all images in the matrix or importing only those whose files haven't been created yet.
  • Creation from templates and/or pose lists. The Pose List and Template tabs contain "Add to Matrix" buttons to load a pose list or template into the matrix.
