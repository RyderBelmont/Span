Dialogue Editor

  • An Expand All button has been added.
  • You can now type stage IDs into stage dropdowns.

Bug Fixes

  • Filtering cases by target has been fixed.
  • Filter (+) has been removed from the menus. Use the Filter submenu to pick a specific filter type.
  • Target Tag now converts to a target filter with that tag.
  • Mint Chocolate now has a hover effect for menus in the condition editor.
  • When adding a Target, Also Playing, or Opponent filter, focus will jump to the Character field rather than the Filter field so that you can immediately select a character.
  • Adding multiple Self or Target filters now works out of the box without needing the cumbersome workaround of adding an Anyone filter and changing it to Self or Target.
  • Ranges can now be specified for specific character filters, for targeting whether 0 or multiple of that character is present.
  • Stop directives on looped camera and fade animations now appropriately stop those animations during the preview.
  • Since 5.2, cases were getting a redundant tag attribute. These will be stripped away now.
