Bug Fixes

  • Fixed issue where visiting a case with a misspelled word multiple times before going to the Spell Check tab would mark that word as misspelled multiple times, which would then cause trying to correct the word in Spell Check to instead mess it up.
  • Fixed minor performance issue with filtering characters by status.
  • Fixed issue with time range conditions (ex. Time in Stage) not retaining the first value entered.
  • Fixed crash when clearing the ID field in the Pose Maker.
  • Fixed issue where a blank pivot for a sprite was treated as (left,top) in the epilogue editor but is treated as (center,center) in game.
  • The validator no longer complains about missing poses for hidden lines.
