Deck Designer

  • Adds the ability to create custom card decks. Go to the "Decks..." button in the toolbar to create a new deck.

Dashboard

  • The New Incoming Lines graph now also identifies lines from new characters that were added since last launch.

Collectibles

  • Adds settings related to the new Wearable Collectibles feature.

Settings

  • Adds an option in the Settings window to turn off the Validator's blank-lines check (under Dialogue). Generally, don't disable it unless you're specifically working on a mute character.

Validator

  • Flags cases that have no lines assigned to them.
  • Flags cases that depend on unused markers.
  • Flags variable tests with no subject.
  • Flags lines that use {small} for the entire line (they should be using the text-size option instead).
  • Flags use of {!reset} at the end of a line.
  • Notifies if the character has no clothing classifications.
  • Notifies that layers like "panties" or "glasses" should be set to plural.

Bug Fixes

  • Fixes the Text Size per-line dropdown sometimes getting "stuck" on a certain value.
  • Custom poses that take files from a different character's folder should now work again.
  • Restores the green "targeted line" highlight for lines targeting the human player, if that setting is turned on.
  • The Validator no longer warns for Pose Maker poses that exist for the base costume but are missing in an alternate costume, because those work automatically now.
  • The Validator no longer warns for completely blank lines in hidden cases.
