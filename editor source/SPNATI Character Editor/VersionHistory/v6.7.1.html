Deck Designer

  • Adds the ability to create custom card decks. Go to the "Decks..." button in the toolbar to create a new deck.

Bug Fixes

  • Fixes the Text Size per-line dropdown sometimes getting "stuck" on a certain value.
  • The Validator no longer warns for Pose Maker poses that exist for the base costume but are missing in an alternate costume, because those work automatically now.
  • The Validator no longer warns for completely blank lines in hidden cases.
