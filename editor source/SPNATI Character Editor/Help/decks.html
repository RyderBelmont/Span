Decks

Overview

Custom decks allow the player to personalize their experience. They can pick and choose cards to play with from any deck they have unlocked.

Card Images

To create a deck, you will need to create images for your cards. The editor will not help you with this. Cards should have a 3:4 aspect ratio and their file names should be in the form <suit><rank>.png from 1-13 for each suit. Examples:
File Name Card
heart2 2 of Hearts
clubs8 8 of Clubs
diamo12 Queen of Diamonds
spade1 Ace of Spades

These images should be placed in a folder under the img directory. If desired, you can make different sets of card fronts for the same deck in the same folder by using a NUMBERLESS prefix (ex. a_heart1.png, b_heart1.png). It's possible to make partial sets (ex. only hearts and diamonds) or partial ranks (ex. only face cards).

Creating a Deck

In the toolbar, go to Decks... and type the key for your new deck. The name of the folder you put your images in is a good choice. Click Create to create the new deck.

Editing a Deck

In the toolbar, go to Decks... and search for a previously created deck. Deck Properties:
  • Title: The name of the deck as it will appear in game
  • Subtitle: A subtitle for the deck as it will appear in game
  • Description: A brief description or flavor text for the deck
  • Credits: Who made the artwork
  • Character/Collectible: You can tie unlocking a deck to unlocking a particular collectible by specifying a character and collectible here. If blank, the deck will be available by default.
  • Status: Where this deck should be available. Blank means it's available in all versions of the game where custom decks are allowed.
  • Card Back: Use + and - to add or remove backs. Click ... to navigate to an image to use for the selected card back. This image should be inside the img folder.
  • Card Front: The editor will locate any folders within the img folder that it thinks contains deck images (based on the naming conventions above). Use the magnifying glass to find your set of images. Use the + to add another set of fronts.
