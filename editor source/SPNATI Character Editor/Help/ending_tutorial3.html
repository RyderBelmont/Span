Epilogue Tutorial 3 - Fades
This tutorial assumes you have completed Parts 1 and 2, since it will build off of the same epilogue. For this tutorial, we'll be adding a fade effect to the orb, to give it an explosion.

Flash Effect

First we'll add a flash effect to the orb's explosion.
  1. Open up your epilogue and select the last directive in the Scene: Blue scene. This should be a "Pause".
  2. Add a "Clear Speech Bubbles" directive.
  3. Add a new sprite (Add Directive > Add Sprite). Use ep_flash.png from Florina's folder.
  4. Give this sprite an ID of "flash".
  5. Drag the flash to the center of the orb, around (X: -335, Y: 1176).
  6. Find the Opacity (0-100) in the Property Table and set it to 0 so the flash starts off invisible.
  7. Set Scale X: 1 and Scale Y: 1. These are the default values, but we're going to animate them so we need to set them explicitly.
  8. Add an "Animate Sprite/Emitter" directive.
  9. Set the ID to "flash" to associate with the flash sprite.
  10. Set the animation's time to 0.2.
  11. Put the mouse over the corner of the flash's box in the preview canvas and drag it to scale the flash. Alternatively, manually set Scale X: 5 and Scale Y: 5.
  12. Drag the opacity slider to full, or type in 100.
  13. Add a key frame.
  14. Set Time: 0.4, Scale X: 1, Scale Y: 1, and Opacity: 0.
  15. Preview the animation. The flash should grow quickly and disappear. This is easiest to see if you have the @0.4s key frame selected.

Vanishing orb

The orb exploded, so it should go away. This section isn't doing anything we haven't already covered, so it'll be brief.
  1. Add an animation directive for ID: orb.
  2. Set Time: 0.2, Delay: 0.2, and Opacity: 0.
  3. These settings will cause the animation to play 0.2s in, coinciding with when the flash is at its largest point, and finish after 0.2s, coinciding with the end of the flash.
  4. Play through the scene and verify everything looks correct. By the end, only Florina should be left.

Screen Fade

To add to the flash's intensity, we'll make the whole screen go bright yellow.
  1. Add a "Fade Effect" directive.
  2. Set the opacity to 100. The whole screen goes black.
  3. Set Time: 0.3.
  4. In the color field, type FFFFC6. This method lets you type exact RGB hex codes rather than using the color picker.
  5. Add a key frame to the fade effect.
  6. Set Opacity: 0.
  7. Play through the scene. The whole screen should flash now.

Clothing Damage

As required when using summoning orbs in the world of SPNATI, its explosion must destroy clothing. We'll do this by swapping out Florina's image with another one.
  1. Select the Add sprite (sprite1) directive at the start of Scene: Blue. Change the ID to "florina" so that it's easier to tell at a glance what we're animating.
  2. Select Scene: Blue and add an "Animate Sprite/Emitter" directive. This will add it to the bottom of the scene.
  3. Set the ID of the animation directive to "florina"
  4. Under the Source field, click the ... and choose 6-embarrassed.png from Florina's folder.
  5. Florina is a bit stretched out. This is because the old image and new one aren't the same width.
  6. It's generally better to make your source images the same dimensions for situations like this, but since Florina's assets weren't designed with epilogue tutorials in mind, we'll just work around it.
  7. Grab the left or right edge of Florina's box and pull it in until she looks good. Alternatively, manually set Scale X: 0.70.
  8. Finally, so that her clothes don't disappear until the flash, set the Delay to 0.3 seconds.
  9. Select the Animate (florina) line again in the Scene view and press Ctrl+Space. This will add a "Wait for Input" pause after it. Every directive type has a keyboard shortcut so you don't have to go through the Add Directive menu.
