Case Templates

Introduction

Case templates are a way to add your own categories to the Dialogue Tree. You define a case type and some default conditions, and from then on you'll be able to create new cases using those conditions from the dialogue tree. Any cases that fit under the template will appear beneath its category.

Managing Templates

You can create or delete templates using the Case Templates Editor.
  1. To open the editor, go to Tools > Manage Case Templates...

Template Editor

If you've written dialogue before, the template editor will look very familiar. It's exactly the same thing as the Conditions table for targeted dialogue in the Dialogue Editor.
  1. Let's say we want a gender-neutral "Removed Accessory" case type.
  2. On the Templates tab, click the + to create a New Template.
  3. Change the type to "Opponent Stripped".
  4. For the label, type "Opponent Removed Accessory".
  5. For the group, click the magnifying glass and look for an appropriate group. Choose "Opponent removing major".
  6. In the Conditions section, click Target > Clothing (Type).
  7. In the Clothing (Type) field that appears, enter "major".
  8. In the Lines section, enter "This is my default text."
  9. Click the "Case Templates" tab.

Using Templates

From now on, the template will appear in the dialogue tree.
  1. Open a character and go to the Dialogue tab. Click the arrow beside the plus.
  2. Under the "Opponent removing major" menu item, you will find an "Opponent Removed Accessory" option.
  3. Click this option to create a new case.
  4. A new case is added to the dialogue tree beneath an "Opponent Removed Accessory" section with the clothing type == major condition already filled in, and a default line saying, "This is my default text."
