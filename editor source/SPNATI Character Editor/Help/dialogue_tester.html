Overview

The Dialogue Tester serves as a sort of Game Simulator. It allows you to set the exact state of the game (which characters are playing, their stages of undress, who lost the round, etc.) in order to see which dialogue lines are selected under certain conditions. This is particularly helpful for verifying that targeted dialogue is set up properly, which to test in game requires a good amount of luck that the cards will fall your way, or through debugging mods.

The Dialogue Tester is found in the main toolbar.

The Tester does not fully simulate the game.
The tester may be out of date with respect to the game code. It doesn't include the player character, meaning that totalMales conditions etc. will be off. It is recommended that you don't rely on it too much but also test your character in the actual game, using the DEBUG mode.

Character State

The tester will open with your character and three random others. You can set each character's state individually:
  • Character Sets which character is occupying this slot.
  • Stage Sets the character's stage of undress.
  • Hand What hand the character has.

Game State

The current game state is controlled at the very bottom.
  • Game Stage: This controls what case should be used. Game stages include:
    • Selected: When selecting the characters.
    • Start: Start of a game after selecting the characters.
    • ExchangingCards: Cards are being exchanged.
    • GoodHand: Characters got a good hand while waiting for the player.
    • OkayHand: Characters got an okay hand while waiting for the player.
    • BadHand: Characters got a bad hand while waiting for the player.
    • BeforeLoss: After the cards have been revealed and a loser has been decided.
    • DuringLoss: The loser (target) is starting to strip or masturbate.
    • AfterLoss: The loser has finished stripping or masturbating. Note that for the target this will actually use the clothing stage following the one that is selected.
    • HeavyMasturbating: The target character is close to finishing masturbating. The character must be in the Masturbating stage for this to work.
    • Finishing: The target character is finishing masturbating. The character must be in the Masturbating stage for this to work.
    • Finished: The character has just finished masturbating. The character must be in the Finished stage for this to work.
    • GameOver: After all players have been eliminated and a winner has been decided. The target is the winner.
  • Target: For game stages that have targets, this determines who the target is.
  • Filter: This will pretend the target has a tag even if they don't. Useful for testing targeted dialogue that uses filter tags without knowing which character actually has the tag.
  • Repeat: This randomly selects new lines so you can see multiple ways a certain game state can play out.
