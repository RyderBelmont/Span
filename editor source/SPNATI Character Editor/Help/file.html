New

Use this option to create a new character. You will be presented with a popup to choose a name. This is the name of the folder that will go in the opponents directory.
  • If you have already created this folder and generated images beforehand, the folder will be available to pick, and might already be filled in.
  • If you are creating the character completely from scratch, just type an appropriate name.

Open

Use this option to edit an existing character. The editor will automatically open to whatever character was last saved.
Why isn't my character showing up?
If your character isn't in the list, one of their xml files is likely invalid. Check errorlog.txt for more details. You should never run into this issue when editing characters exclusively in the editor, but this can happen if people manually edit the xml.

Save

Use this option to save your changes. This will save data from all tabs besides the Images tab.

When saving, the editor will regenerate placeholder lines for any missing tags.

Auto-backup
The first time you save each day, the editor will create a backup of meta.xml and behaviour.xml. These can be found in your character's folder with the .edit.bak file extension.

Save As

When using this option, you can choose a new folder name to save your character to.
I did this and now my images are blank! What gives?
Save As will only copy the behaviour.xml and meta.xml files to the new folder, not any images, which admittedly makes this option not very useful currently.

Import .txt

Use this option to pull dialogue from a text file. This works essentially the same way that make_xml.py does.
This will replace ALL data besides images. If you aren't careful, you could lose work. For example, if you add lines in the editor and then import a text file that doesn't contain those lines, they will be lost.

Export .txt

Use this option to export your character data (excluding images) into a text file. This text file can be run through make_xml.py to regenerate dialogue, or you can import it back into the editor later. This option is primarily for collaboration purposes. If you are making a character on your own, there's little reason to work outside the editor.

That said, many experienced creators prefer working with a text file, so when you are ready to submit your character, you should export a text file as well.

Exporting will create or overwrite edit-dialogue.txt in your character's folder.

Setup

Opens the Setup window that appeared the first time you launched the Character Editor.

Exit
