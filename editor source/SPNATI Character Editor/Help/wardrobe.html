Overview

Your character will strip in a pre-determined order, which never changes from game to game. The Wardrobe tab is where you define this order.
Each layer of clothing contains extra information about that article that other players will react to - what it covers, how much it covers, etc.

This screen lists out the layers of clothing your character has, ordered in the same order that your character will remove them in.

To begin, click the Add button to create a New Item, the first layer that will be stripped.
The list still says "New item" after I rename it.
The list doesn't update immediately, but rest assured that your item is renamed. As soon as you add another layer, the name will update in the list.

Setting Layers

Strip order can be controlled using layers.
  • Adding a Layer: Use the Add button to generate a new layer. There is no limit to layers, but no existing character has more than 8.
  • Removing a Layer: Use the Remove button to delete the selected layer.
  • Reordering a Layer: Use the up and down arrows to move a layer around in the strip order.
Be careful changing layers after you've already written dialogue.
Rearranging layers will also rearrange that layer's (stage's) dialogue. While you won't lose any lines (unless you delete a layer, of course), it can be a real pain to update dialogue to reflect the new strip order.

Layer Data

Each layer has a few relevant pieces of information.
  • Proper Name: Use the Add button to generate a new layer. There is no limit to layers, but no existing character has more than 8.
  • Lowercase Name: Use the Remove button to delete the selected layer.
  • Type: How much the clothing covers. Stripping reactions are based on this type.
    • extra: Accessories or other minor, non-clothing articles.
    • minor: Clothing that covers some skin, but not much. Examples: outer jacket, socks, etc.
    • major: Clothing that covers a lot of skin, usually revealing underwear. Examples: shirt, dress, pants, etc.
    • important: Clothing that covers a vital body part (chest or genitals). Typically underwear.
  • Position: Where the clothing is located. This is only used for determining if an important layer is revealing a character's crotch or chest.
    • upper: Covers the character's chest
    • lower: Covers the character's crotch
    • other: Covers some other part of the character
