Setup

The first time you run the Character Editor, it will ask where you have downloaded the GitLab source to. Navigate to the root level of your game folder (i.e. the one containing the js, css, opponents folders, etc.). The editor needs to know this information so it knows where to read character information from, among other things.

Creating a New Character

Once you've finished setup, you are ready to create a new character. Click Characters... to bring up a list of all recognized characters. Type the name of the folder you want to give your character and press enter. Assuming there isn't already a folder of that name, this will enable the "Create New" button. Click this to create your character.
