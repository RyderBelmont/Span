Adding Your Character to the Game

Testing Your Character In-Game

Once you're ready to try out your character in game, go to the Metadata tab and click "Add to Listing." This will make your character available for character selection in the offline version of the game.
For Private Testing Only
While this option does the same thing as the steps necessary to actually add the character to the final game, this is meant strictly for private testing purposes. In short, don't commit listing.xml when you are pushing your character's files to the repository.

Submitting Your Finished Character

When your character is finished, you still need to go through the standard process of putting them on the testing tables and getting people to vouch for the character's readiness.

Submission Checklist

  • Run the validator on your character and make sure there are no warnings.
  • Export your character to a text file. (File > Export). Commit this edit-dialogue.txt (and rename it as appropriate)
  • Make sure you have a saved pose list for your kisekae images. Commit this text file.
  • Commit all your images.
  • Commit behaviour.xml and meta.xml.
  • Do not commit backup files (.bak extension).
  • Push these to local fork, and then submit a merge request as outlined on the subreddit.
