Anonymous Character Usage Tracking

We would like to collect anonymous information to help make the game better and more enjoyable for everyone.

The exact information collected includes:

  • Your browser type and operating system
  • The website you are accessing this game from
  • Your selected player gender and size
  • The characters you play with
  • Any game script errors you encounter
  • The current date and time

We would like to emphasize that this information will be completely anonymous: we won't be able to identify you using this info.
